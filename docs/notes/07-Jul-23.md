### 06-July-23

无符号编码 unsigned 用于编码无符号整数
补码 two's-complement 常用于整数的编码
浮点数编码 floating-point 科学计数法的编码

计算机使用有限的位编码数字，结果不足以表示时运算会溢出overflow

浮点数编码是近似的，不是精确的

大多数计算机使用8位的块作为最小的可寻址的内存单位，也就是字节`byte`。机器级程序将内存视为一个非常大的字节数组即虚拟内存virtual memory。内存的每个字节都有唯一的数字标识，称为它的地址address。所有可能地址的集合就是虚拟地址空间virtual address space

!!! tip
    64位OS一般是指操作系统的字长`word size`为64位，指明指针数据的标称大小normal size。虚拟地址是以这样的一个字来编码的，所以字长决定的最重要的一个系统参数就是虚拟地址空间的最大大小。
    对于一个字长为ω的机器而言，虚拟地址范围为$\ 0 \sim  2^ω-1$

#### 进制转换

当十进制数$\ x$满足$\ x=2^n$时，可以很容易将$\ x$写成十六进制数
$\ x$ to bin: 10000... (n个0)
十六进制0代表4个二进制0
$\ n = i + 4j (0 \le i \le 3)$。即$\ x = 2^n = 2^{i+4j} = 2^i * 2^{4j}$

i = 0 ... 1(0000)*j ... 2^0

i = 1 ... 2(0000)*j ... 2^1

i = 2 ... 4(0000)*j ... 2^2

i = 3 ... 8(0000)*j ... 2^3

当十进制数不满足特殊形式时，可以反复除法

314156 to hex:

$\ 314156 = 19634*16+12$ .... C

$\ 19634 = 1227*16+2$ .......... 2 

$\ 1227 = 76*16+11$ ............. B

$\ 76 = 4*16+12$ .................. C

$\ 4 = 0*16+4$ ...................... 4

314156 to hex: 0x4CB2C
